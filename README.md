# Level Up Daily

PWA de hábitos + gamificación con Flappy Bird integrado.

## Stack
- React 19 + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- localStorage
- Capacitor (Android)

## Ejecutar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Android (Capacitor)
```bash
npm run android:setup
npm run android:update
```
